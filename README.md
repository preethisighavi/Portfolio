## How to Deploy (GitHub Pages)

1. Create a repo named `preethisighavi.github.io`
2. Push all files to the `main` branch
3. Go to **Settings → Pages → Source → main / root**
4. Your site goes live at `https://preethisighavi.github.io`

---

## Customization Notes

- **Resume:** Drop your PDF into `assets/PreethiSighavi_Resume.pdf` — the download button is already wired up.
- **Colors:** All CSS custom properties live at the top of `styles.css` under `:root { }`.
- **New projects:** Add a new `<article class="project-card">` block inside the `#projects` section in `index.html`.
- **Profile photo:** Add an `<img>` in the hero or about section pointing to `images/profile.jpg`.
